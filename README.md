## 关于模式选择


	1.橙武模式（[雾海寻龙]橙武花间bate1.mm）
		自写的橙武循环，只打5豆以上的墨海
	2.紫武模式（[雾海寻龙]紫武花间bate1.mm）
		白皮书的紫武循环，只在芙蓉循环和乱洒循环打墨海
	3.MFC模式（[雾海寻龙]MFC花间bate.mm）
		MFC的一键宏，在此基础加了自动腰坠，自动切宏；如果你不知道这是什么请前往www.jx3box.com/macro/75801查看

	--如果你是大橙武，可选有以下模式
		1.橙武
		2.MFC
	--如果你不是大橙武，可选有以下模式
		1.紫武
		2.MFC

### 三、进行取图

| 取图项 | 取图说明 | 
| :----: | :---- | 
| 快雪 | 读条完快雪，图标亮起后框选内部`左上角1/4` | 
| 流离 | 释放阳明指，图标亮起后框选图标`内部任意` | 
| 商阳 | 释放商阳指，图标亮起后框选图标`内部任意` | 
| 溅玉 | 读条完快雪，图标亮起后框选内部`左上角1/4` | 
| 橙武 | 取消隐藏，图标亮起后取框选图标`内部任意` | 
| 5豆 | 打5个阳明指，图标亮起后框选图标`内部任意` | 
| 8豆 | 打8个阳明指，图标亮起后框选图标`内部任意` | 
| 墨海1层 | 打一次墨海，使图标的数字变成1，框选`数字1` | 
| 墨海0层 | 打两次墨海，使图标的数字不显示，框选内部`右下角1/4` | 
| 水月 | 水月好的时候，框选图标`内部任意` | 
| 芙蓉 | 芙蓉好的时候，框选图标`内部任意` | 
| 乱洒 | 乱洒好的时候，框选图标`内部任意` |

> [!TIP]
> 取图完成后在茗伊插件→目标监控里将需要隐藏的监控，把“隐藏消失的”选项勾选上。

> [!TIP]
> 截图buff图标时，这里更推荐你截左上角的位置，尽量避开右下角，因为右下角是有层数数字的，这会导致识别错误

> 如果使用紫武和MFC的bate版本，找不到取色按钮的就是不用取(监控使用的是一样的，有些用不到)

> 因为游戏问题，墨海的监控在重新上线后不显示充能层数，需要使用一次墨海技能才会显示出数字层数  
> （如果你的墨海临源监控不显示层数数字，那么是不会打墨海技能的）

> [!TIP]
> 注意：水月/芙蓉/乱洒这3个监控在战斗中会变成2个图标，请不要让出现的第2个图标遮挡其他图标

 ### 六、如何起手？
 

墨海/芙蓉/水月/乱洒图标是亮起的；其他图标没有亮起；你的下一个兰摧是需要读条的，那么你可以：
 
 	按取色按键开启取色-预读条兰摧-读条过程中按开关启动按键
  
如果你看不懂上面的情况，那么你可以

 	按取色按键开启取色-按开关启动按键

 ### 七、功能按钮说明
 
 说明：
 
 	开始运行：开始运行这个职业
 
	模式键：选择【橙武取色】【紫武取色】【MFC取色】开始取图取色（开始取图后按开关才能开启和暂停，没有开启取图是不会按任何键的）
 
	总开关键：暂停/启动按键
